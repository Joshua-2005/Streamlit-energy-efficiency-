# ⚡ Energy Efficiency Prediction App

A modern **Machine Learning web app** built with **Streamlit** that predicts energy efficiency based on input features and automatically selects the best model.

---

## 🌐 Live App

👉 https://energy-efficiency-ml-app.streamlit.app

---

## 🚀 Features

* 📂 Upload datasets (CSV / Excel)
* 🤖 Automatic model comparison
* 🏆 Best model selection
* 📊 Visual performance comparison (bar chart)
* 🔮 Real-time prediction system
* 🎨 Modern UI with custom background

---

## 🧠 Machine Learning Models Used

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

---

## 📊 How It Works

1. Upload your dataset
2. Select target column (output)
3. Select feature columns (inputs)
4. App trains multiple models automatically
5. Best model is selected based on accuracy
6. Enter values → Get prediction instantly
